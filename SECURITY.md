# Security Policy

## Supported Versions

Security updates are currently provided for the latest `main` branch.

## Reporting a Vulnerability

Please do not open public GitHub issues for security reports.

Email: **arslanbasharat.o.o@gmail.com**

When reporting, include:

- Affected endpoint or file path
- Reproduction steps
- Impact assessment
- Any proof-of-concept details

We aim to acknowledge reports within 72 hours and provide status updates until resolution.
